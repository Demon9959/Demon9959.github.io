Welcome to Lab 3 – SEO & Indexing Test Page

This site was created as part of PR Lab 3 to demonstrate how websites are indexed by search engines such as Google. It includes correct usage of meta tags, sitemap.xml, robots.txt, and internal linking. Search engine optimization (SEO) is essential for ensuring that important content is discoverable and properly ranked.

Topics illustrated here include:

  • Basic SEO optimization using <meta> tags
  • Google Search Console verification
  • Indexing control using robots.txt
  • Discovery of hidden pages via Google Dorks

This page is intentionally made visible to demonstrate proper indexing. If you’re seeing this from a Google search, indexing is working!
