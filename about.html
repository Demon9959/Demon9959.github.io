About This Project

This web project is a part of Lab 3 in the PR course. The goal is to learn how websites are discovered and indexed by search engines using modern tools such as:

  • Google Search Console
  • robots.txt file and its usage
  • Meta tags like robots and description
  • Sitemap.xml creation and submission

By building this site with multiple pages and varying access permissions, we simulate how public and private content is handled on the web.
